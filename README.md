# 如果电话亭 What-If Phone Booth

文旅策划创新组合工具。不是助手，是道具——拿起来就用，用完就放。

> 输入：项目地 + 项目类型
> 输出：一张可直接用的方案卡片

## 核心公式

```
项目地 A（地方特征） + 项目类型 B（体验形式） = "如果A做B..."
```

## 安装

把 `what-if-phone-booth/` 目录放到 Claude Code 的 skills 目录下：

```bash
# 全局加载（所有项目可用）
cp -r what-if-phone-booth ~/.claude/skills/

# 或项目级加载（仅当前项目可用）
cp -r what-if-phone-booth /your/project/.claude/skills/
```

## 使用

### 方式一：slash command

```
/what-if-phone-booth 通州三庙一塔 夜间水面体验
```

### 方式二：自然语言触发

直接说：
- "启动，如果电话亭"
- "如果电话亭，景德镇陶瓷，森林夜游"
- "帮我组合一下"
- "嫁接" / "文旅创新组合"

## 五步法工作流

| Step | 做什么 | 替代了什么 |
|------|--------|-----------|
| 1. 深度挖掘 | 8维度文化素材提取（每个维度3-5个关键词） | 花3天查资料 |
| 2. 案例检索 | 检索前沿沉浸式/文旅案例 | 花2天找对标 |
| 3. 组合生成 | 5种策略 × 多样性强制约束 → 3-5个方案 | 花1周头脑风暴 |
| 4. 标准化输出 | 如果句 + 组合逻辑 + 关键场景 + 可视化设计要素 | 花2天写PPT |
| 5. 询问保存 | 询问是否保存 + 路径 | 手动整理归档 |

## 多样性强制约束

生成方案时，任意两个方案之间**至少3个维度不同**：

| 维度 | 必须拉开的方向 |
|------|---------------|
| 感官通道 | 视觉 / 听觉 / 触觉 / 嗅觉 / 味觉 / 温度感 |
| 载体空间 | 水面 / 陆地 / 建筑立面 / 空中 / 人体穿戴 / 地下 |
| 情绪光谱 | 震撼 / 宁静 / 幽默 / 恐惧 / 仪式感 / 童趣 / 怀旧 / 好奇 |
| 参与深度 | 纯观看 / 可触摸 / 可行走穿越 / 需对话共创 / 需集体协作 |
| 时间节奏 | 瞬间爆发 / 缓慢生长 / 需要等待 / 有仪式流程 / 可逆/不可逆 |

## 方案卡片模板

每个方案输出包含6个部分：

1. **如果句** — 一句话概括核心概念（PPT标题页直接用）
2. **组合策略** — 使用了哪种策略（方法论标签）
3. **组合逻辑** — 为什么这样组合（说服甲方的论据）
4. **关键场景** — 核心体验场景描述（给导演/空间设计师）
5. **设计亮点** — 差异化卖点
6. **可视化设计要素** — 空间类型 / 材质语言 / 光影策略 / 色彩体系 / 形态关键词 / 效果图Prompt

## 理论来源

- Arthur Koestler《创造的行为》— 组合创新理论
- Miller's Law（7±2）— 信息容量约束
- CCGS Question-First — 协作协议
- 什克洛夫斯基 — 陌生化（Defamiliarization）
- Gibson — 功能可见性（Affordance）
- 渐进式披露（Progressive Disclosure）— 架构设计
