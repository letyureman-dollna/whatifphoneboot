# 线下体验产品形态库

Phase 2 组合时从以下形态中选择最匹配当前项目类型的 3-5 种作为组合载体。

---

## 1. 沉浸式演艺

| 形态 | 核心特征 | 典型案例 |
|------|---------|---------|
| 沉浸式戏剧 | 观众自由行走、多线叙事、演员与观众零距离 | Sleep No More、只有河南·戏剧幻城 |
| 环境戏剧 | 非传统剧场空间、建筑即舞台 | 陶溪川窑炉演出 |
| 夜游演艺 | 户外夜间演出、灯光+投影+实景 | 长恨歌、印象系列 |
| 无人机灯光秀 | 无人机编队空中表演、大规模视觉奇观 | 重庆无人机灯光秀（90场/年）|
| 水秀/喷泉秀 | 水幕投影+音乐+灯光 | 千岛湖水秀 |
| 烟花秀 | 烟火编排+音乐同步 | 迪士尼烟花 |

## 选择指南

| 项目类型 | 优先匹配的形态 |
|---------|--------------|
| 文旅综合体 | 沉浸式演艺 + 沉浸式空间 + 市集 + 空间改造 |
| 夜游项目 | 夜游演艺 + 灯光装置 + 气味景观 + 露天电影 |
| 博物馆升级 | 体验式博物馆 + 数字沉浸 + 互动技术 |
| 商业改造 | 快闪展览 + 创意市集 + 沉浸式餐饮 |
| 工业遗址 | 空间改造 + 沉浸式戏剧 + 充气装置 + 大地艺术 |
| 社区营造 | 社区花园 + 口袋公园 + 市集 + 公共艺术 |
| 节庆活动 | 无人机灯光秀 + 烟花秀 + 美食节 + 快闪 |
| 微度假 | 露营 + 树屋 + 农场体验 + 徒步路线 |
